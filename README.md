# GomyCode Cybersecurity Bootcamp — Checkpoints

This repository documents all checkpoint submissions from my cybersecurity bootcamp at **GomyCode**. Each checkpoint covers a core topic area in cybersecurity, progressing from networking fundamentals through to network defense, threat analysis, and data protection strategies.

---

## 📁 Repository Structure

```
gomycode-cybersecurity-checkpoints/
├── checkpoint-1/   → Networking Fundamentals & Lab Setup
├── checkpoint-2/   → Security Fundamentals (CIA, AAA, Cryptography)
├── checkpoint-3/   → Threat Actors, Malware & Social Engineering
├── checkpoint-4/   → Network Security, Cloud & Data Protection
├── checkpoint-5/   → (Image-based submission — see notes)

```

| # | Topic | Key Concepts |
